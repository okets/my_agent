# M9.2-S9: Skill Filter Wiring — Sprint Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire the `filterSkillsByTools()` return value into prompt assembly so skills requiring unavailable tools are actually excluded. Completes the S8 skill filter refactor.

**Why:** S8 made `filterSkillsByTools()` a pure function (no disk writes, crash-safe). But nothing uses the result — skill filtering is computed but never applied. The function identifies which skills to disable, then the result is discarded. This step adds the consumer.

**Prerequisite:** S8 merged (skill filter is a pure function returning disabled skill names).

---

### Step 1: Add `excludeSkills` parameter to `assembleSystemPrompt()`

- [ ] **1a: Modify `assembleSystemPrompt` signature**

In `packages/core/src/prompt.ts`, add an optional parameter:

```typescript
export async function assembleSystemPrompt(
  brainDir: string,
  options?: { excludeSkills?: Set<string> },
): Promise<string> {
```

In the framework skills scan loop (currently lines ~604-618), skip entries whose name (without `.md`) is in the exclude set:

```typescript
for (const entry of entries) {
  if (!entry.endsWith('.md')) continue
  const skillName = entry.replace(/\.md$/, '')
  if (options?.excludeSkills?.has(skillName)) continue
  // ... existing loading logic
}
```

### Step 2: Wire callers

### Step 3: Unit tests

- [ ] **3a: Test exclusion works**

Add a test to `packages/core/tests/prompt-always-on.test.ts` (or a new file):

```typescript
it("excludes skills in the excludeSkills set", async () => {
  const prompt = await assembleSystemPrompt(brainDir, {
    excludeSkills: new Set(["visual-presenter"]),
  });
  expect(prompt).not.toContain("Visual Expression");
  expect(prompt).toContain("Message Routing"); // other skills still loaded
});

it("loads all skills when excludeSkills is empty", async () => {
  const prompt = await assembleSystemPrompt(brainDir, {
    excludeSkills: new Set(),
  });
  expect(prompt).toContain("Visual Expression");
  expect(prompt).toContain("Message Routing");
});

it("loads all skills when excludeSkills is undefined", async () => {
  const prompt = await assembleSystemPrompt(brainDir);
  expect(prompt).toContain("Visual Expression");
  expect(prompt).toContain("Message Routing");
});
```
