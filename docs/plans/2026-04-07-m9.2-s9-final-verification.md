### Step 4: Delegation tests (S6 reruns)

- [ ] **4a: Forced simple delegation**

Open dashboard, send:
> "Check the memory usage of this machine 2 minutes from now and tell me the results."

Verify:
1. Nina calls `create_automation` (NOT inline, NOT `create_task`)
2. Tool call succeeds — `todos` array present (S4 schema enforcement)

5. Job fires, worker completes
6. `todos.json` has delegator items + generic template items
7. `status-report.md` exists and passes validator
8. Worker prompt does NOT contain "You do not do work yourself" (S8 fix)

- [ ] **4b: Complex research delegation**

Open dashboard, send:
> "Find out what the top 3 rated Thai restaurants in Chiang Mai are. Compare their ratings and price ranges."

Verify:
1. Nina delegates via `create_automation` with `job_type: "research"`
2. `todos` array has multiple task-specific items
3. Job runs to completion
4. Worker's `todos.json` has all 3 layers:
   - Layer 1: Nina's task-specific items
   - Layer 2: Research template items (sources, cross-check, chart, status-report)
   - Layer 3: Any worker-added items
5. Worker called `create_chart` for comparison data (or noted why not)
6. `status-report.md` includes sources list
7. All mandatory items are `"done"`
